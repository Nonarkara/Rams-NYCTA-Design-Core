§2 Typography — Helvetica Neue lineage tabular figures · weight ceiling 600 · tracking scales inversely with size

NYCTA §6 The Arrow solid equilateral triangle · CSS-drawn · no unicode
ALL MARKETS

CSS: border-top:4px solid transparent;

The arrow takes trunk color ONLY when it points to a specific board. Grey by default (directional, not identity). Single solid triangle. Never double-arrows, outlined arrows, or rounded arrowheads.
Decision Tree — Should This Be Colored? run top to bottom · stop at first match
DATA? Value whose state matters (up/down/live) → bare --accent / --neg. No disc.
IDENTITY? Which board / family / route → enclose it. Disc or station-plate rule, trunk color from §3.
DIRECTION? Go here / more → greyscale solid triangle. Color only if pointing to one specific board.
NONE? No color. Use grey + size. If you're adding color "to liven it up" — delete it.
