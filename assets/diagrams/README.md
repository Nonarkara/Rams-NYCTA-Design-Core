# Diagram Assets

**Drop diagrams here.**

## Naming Convention

```
[type]-[subject]-[version].svg
```

Examples:
- `arch-system-overview-v1.svg`
- `flow-data-pipeline-v2.svg`
- `map-regional-coverage-v1.svg`

## Types

| Type | Use |
|---|---|
| `arch-` | System architecture diagrams |
| `flow-` | Process / data flow charts |
| `map-` | Geographic or conceptual maps |
| `wire-` | Wireframes and layout sketches |
| `chart-` | Standalone charts exported from data |

## Rules

- SVG preferred. PNG at 2× only when SVG is not practical.
- Use token colors only: `--ink`, `--ink-2`, `--ink-3`, `--line`, `--accent`, `--neg`, trunk colors.
- No gradients, no drop shadows, no decorative fills.
- Line weight: 1px for hairlines, 2px for structural, never thicker.
- Labels: font-size 11–12px, weight 500–600, UPPERCASE for category labels.
