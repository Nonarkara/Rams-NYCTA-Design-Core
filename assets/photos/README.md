# Photo Assets

**Drop photos here.**

## Naming Convention

```
[context]-[subject]-[number].jpg
```

Examples:
- `hero-bangkok-skyline-01.jpg`
- `field-trading-floor-01.jpg`
- `city-tokyo-data-01.jpg`

## Context Tags

| Context | Use |
|---|---|
| `hero-` | Page hero / cover images |
| `field-` | Field work, real-world operations |
| `city-` | City or urban environment shots |
| `team-` | People and team |
| `event-` | Conferences, presentations, public moments |

## Rules

- Real photos only. No stock filler, no AI-generated images.
- No duplicates across sections of the same site.
- JPEG, 85%+ quality. Width 2400px minimum for hero use.
- Alt text required — describe what is happening, not what it looks like.
- For hero images: apply the legibility overlay `background: linear-gradient(to bottom, rgba(25,23,18,0.5) 0%, transparent 60%)` for text placed on top. This is the only permitted gradient in the system.
- Credit where required by license.
